# Лабораторная работа №1

## Задача
Программа генерирует множество случайных чисел размера n в диапазоне (min, max),
где n, min, max вводятся с клавиатуры. После чего подсчитывает выводит сумму, 
которая получается следующим образом: все числа, номера которых совпадают с дробной 
частью одного из исходных чисел - вычитаются, все остальные прибавляются.

## Как запустить
1. Установить файлы func.h - файл, содержащий функции, и rand.c - файл, содержащий основную часть программы.
2. Запустить файл rand.c и ввести минимум и максимум чисел, а затем количество гинерируемых чисел.
